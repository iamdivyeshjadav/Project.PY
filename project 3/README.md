# 📌 Overview

The **Student Data Organizer** is a menu-driven Python console application designed to help users manage and organize student information efficiently.

This project demonstrates:
- CRUD operations
- Python dictionaries & lists
- Interactive CLI menus
- Match-case logic
- Data organization techniques

---

# 🎯 Project Objective

Build a beginner-friendly student management system using Python fundamentals.

---

# ✨ Features

| Feature | Description |
|---------|-------------|
| ➕ Add Student | Insert student records |
| 📄 Display Students | Show all saved records |
| ✏️ Update Student | Modify student details |
| ❌ Delete Student | Remove records safely |
| 📚 Subjects Viewer | Show available subjects |
| 🔁 Infinite Loop | Runs until user exits |
